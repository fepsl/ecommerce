## Como usar este vault

- Links `[[nota]]` funcionam no Obsidian como hiperlinks entre notas
- Abra o Obsidian → "Open folder as vault" → selecione a pasta `docs/`
- Use `Ctrl+P` para buscar qualquer nota rapidamente
- Use o Graph View para visualizar as conexões entre os temas
